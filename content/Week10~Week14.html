Week10~Week14

WEEK10

google用戶登陸

  • 登錄到您的@gm帳戶
  • 啟用Google的API
  • 按設置同意按鈕按鈕
  • 外部用戶類型
  • 進入“憑據”頁面
  • OAuth 2.0客戶端ID 類型的憑據
  • (設置網址)授權的JavaScript為：https：// localhost：8443
  • 定義URI為：https：// localhost：8443 / login / google /
  • 將oauth_gm.txt保存中
  • .開啟leo更改其中得config中的scrum為自己命名的txt檔案名稱，並且修改完成之後按下save和darwROC
  • pip install authomatic
  • 開啟https://localhost:8443
  • 測試是否可以成功運作

WEEK11

ubuntu設置

操作步驟：

  • 下載vitualbox、ubuntu、CoppeliaSim四輪車控制器檔案。\
  • 開啟vitualbox，匯入ubuntu。
  • 設定虛擬主機的網路埠號。[NET:設定->網路->附加到(A)改成NAT->進階->連接埠轉送->設定三個埠號(22、8443、19999)]。
  • 設定完成後啟動ubuntu。
  • 輸入密碼kmol2020，並開啟lxterminal。
  • 輸入xterm&開啟，並使用ctrl+右鍵將視窗放大。
  • 依照步驟打開coppeliasim，輸入ls查看內容。
  • cd coppeliasim4_rev4->ls->cd CoppeliaSim_Edu_v4_0_0_Ubuntu18_04->ls->./coppeliaSim.sh。
  • 開啟後打開檔案sdc.ttt。
  • 開啟啟動器，並開啟下載好的CoppeliaSim四輪車控制器檔案。
  • 將檔案中的car_model.py放入編譯器中，並執行。
  • 打開CoppeliaSim中的模擬，四輪車開始繞圈即成功。

Week1~Week5

WEEK1

建立個人倉儲cd2020及進行網站編譯

．下載2019Fall可攜套件.7z ，完成後開啟。
．登入個人github帳號，創建新的倉儲cd2020。
．進入要存放資料夾tmp。
．輸入 git clone  https://github.com/s40723201/cd2020  複製個人倉儲至資料夾。
．輸入 git submodule add https://github.com/mdecourse/cmsimde.git  取得子模組資料。
．將 up_dir 的所有項目複製到cd2020。
．完成後，python wsgi.py 開始編輯倉儲。
．編輯完成後，git status -> git add . -> git commit -m "XXX" -> git push。

WEEK 2

將可攜系統中的 python3.7.3 更新至 python3.8.2 版本

．收尋 Python官網。
．點選Downloads下的Windows，並點選 Latest Python 3 Release - Python 3.8.2。
．找到 Windows x86-64 executable installer 並下載。
．開始下載，必須將 pip選項取消

．建立新的start.bat

start.bat:

@echo off
set Disk=y
subst %Disk%: "data"

set PYTHONPATH=%Disk%:\py38\DLLs;%Disk%:\py38\Lib;%Disk%:\py38\Lib\site-packages;
set PYTHONHOME=%Disk%:\py38

REM for flutter
set java_home=%Disk%:\java\jdk8u222-b10
set ANDROID_SDK_home=%Disk%:\home_mdecourse

set ANDROID_SDK_ROOT=%Disk%:\android\sdk

REM for putty

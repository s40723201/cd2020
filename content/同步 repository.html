同步 repository

  1. 顯示目前所有的遠端目錄
git remote -v
  • 應該可看見

origin https://github.com/istar0me/wp106a.git (fetch) origin

https://github.com/istar0me/wp106a.git (push)
  1. 新增遠端 Repository 
git remote add <自行輸入名稱> https://github.com/s40723246/cd2020-bg1.git
    1. 此時確認目前追蹤的遠端清單
    git remote -v
    • 會發現多出兩組 upstream

    <輸入名稱> https://github.com/s40723246/cd2020-bg1.git (fetch)

    <輸入名稱> https://github.com/s40723246/cd2020-bg1.git (push)
    1. 切換到要做更新的分支
    git checkout master
    1. 將組別新版本的檔案拉進自己 master 分支
    git pull <輸入名稱> master
    1. 完成本機端的更新，此時可選擇將新版的檔案上傳到自己的 GitHub 上
    git push origin master
